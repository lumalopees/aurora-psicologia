# Aurora Psicologia

Projeto de portifolio para uma clinica de psicologia com foco em visual profissional, acolhedor e moderno.

## Como rodar

1. Instale as dependencias:
   `npm install`
2. Inicie o servidor:
   `npm run dev`

## Tecnologias

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- next-themes (modo claro/escuro)
